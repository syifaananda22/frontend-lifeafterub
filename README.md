# Frontend Life After UB

Frontend website Life After UB dibuat menggunakan React JS dan Vite sebagai tampilan utama sistem rekomendasi karir mahasiswa dan alumni Universitas Brawijaya.

## Fitur
- Login & Register
- Dashboard User
- Simulasi & Rekomendasi Karir
- Riwayat Simulasi
- Dashboard Admin
- CRUD Data User, Karir, Alumni, dan Rekomendasi Karir

## Teknologi
- React JS
- Vite
- Axios
- React Router DOM
- CSS

## Menjalankan Project

```bash
npm install
npm run dev
