# Frontend Life After UB

Frontend website Life After UB dibuat menggunakan React JS dan Vite sebagai tampilan utama sistem rekomendasi karir mahasiswa dan alumni Universitas Brawijaya. Website ini menyediakan fitur dashboard user dan dashboard admin yang digunakan untuk mengelola data pengguna, simulasi karir, rekomendasi karir, data alumni, serta aktivitas pengguna secara terintegrasi.

## Fitur
- Login & Register
- Dashboard User
- Simulasi Karir, Karir Alumni, Perusahaan Mitra UB
- Riwayat Simulasi
- Dashboard Admin
- CRUD Data User, Karir, Alumni, dan Perusahaan Mitra UB

## Teknologi
- React JS
- Vite
- Axios
- React Router DOM
- CSS
